# RDS (Relational Database Service)

## I. Tổng quan về Amazon RDS

**Nó là gì:**
RDS là dịch vụ database quản lý của AWS, hỗ trợ nhiều engine như MySQL, PostgreSQL, MariaDB, Oracle, SQL Server và Amazon Aurora. AWS lo các công việc như patching, backup, replication, failover. RDS cho phép tạo read replica, Multi-AZ deployment để đảm bảo high availability.

**Khi nào sử dụng:**
* Khi ứng dụng cần relational database mà không muốn tự quản lý server database.
* Khi cần automated backup, point-in-time recovery và Multi-AZ failover.
* Khi muốn dùng read replica để scale read workload.

---

## II. Hướng dẫn cơ bản để bắt đầu

1. Tạo DB subnet group gồm các private subnet trong VPC.
2. Tạo Security Group cho phép kết nối từ application (mở port `3306` cho MySQL/MariaDB hoặc `5432` cho PostgreSQL).
3. Tạo RDS instance qua Console hoặc Terraform, chọn engine, instance class, storage.
4. Bật Multi-AZ nếu cần high availability.
5. Lưu connection string vào AWS Secrets Manager, không hardcode trong code.

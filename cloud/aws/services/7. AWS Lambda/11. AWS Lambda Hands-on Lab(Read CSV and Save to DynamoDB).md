# 11. AWS Lambda Hands-on Lab(Read CSV and Save to DynamoDB) (Lab đọc CSV lưu vào DynamoDB)

Bài thực hành này đã được chuyển sang thư mục hướng dẫn triển khai (**lab**) của dự án để đảm bảo tính phân tách tài nguyên rõ ràng giữa phần tài liệu giới thiệu dịch vụ (Services) và hướng dẫn thực hành triển khai thực tế (Deploy).

Để xem toàn bộ hướng dẫn thực hành từng bước chi tiết (Hands-on Lab) bao gồm:
1. Tạo bảng Students trên DynamoDB với Partition Key `student_id`.
2. Tạo IAM Policy và Role cấp quyền cho Lambda đọc S3 và ghi DynamoDB.
3. Tạo Lambda function Python và viết mã nguồn phân tích tệp CSV sử dụng stream.
4. Cấu hình S3 Event Trigger kích hoạt Lambda khi tải tệp CSV lên bucket nguồn.
5. Tạo tệp CSV kiểm thử và xác nhận dữ liệu tự động lưu vào bảng DynamoDB.
